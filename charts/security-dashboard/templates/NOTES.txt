{{- if eq .Values.scan.mode "node-cache" }}

scan.mode=node-cache: the Trivy scan Jobs mount the node's containerd socket
(hostPath) and require Pod Security Standards "privileged" on this namespace:

  kubectl label namespace {{ .Release.Namespace }} \
    pod-security.kubernetes.io/enforce=privileged --overwrite
{{- else }}

scan.mode=registry: scan Jobs pull image layers from the registries directly.
The namespace can enforce PSS "baseline":

  kubectl label namespace {{ .Release.Namespace }} \
    pod-security.kubernetes.io/enforce=baseline --overwrite

If your workload images live in private registries, give Trivy pull
credentials (see the chart README) or switch to scan.mode=node-cache.
{{- end }}
